# Upgrade Notes

This section provides important notes for upgrading from previous versions of Moil Meeting.

## Upgrading to Latest Version

### Prerequisites

- Ensure you have backed up your configuration files
- Check system requirements for the new version

### Breaking Changes

- None at this time

### Migration Steps

1. Back up your existing installation
2. Download the latest version
3. Follow the installation guide
4. Restore your configuration files

## Version Compatibility

| From Version | To Version | Notes |
|--------------|------------|-------|
| 1.x | 2.x | Full compatibility |
