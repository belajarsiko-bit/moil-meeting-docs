# Welcome to Moil Meeting

**Moil Meeting** is an innovative video conferencing solution leveraging **MoilApp’s high-quality fisheye image processing technology**.

It provides panoramic and wide-angle (360°) views using a single fisheye camera.

---

## Key Features

<div class="grid cards" markdown>

-   :material-camera-iris: **Fisheye Technology**
    ---
    Processing 360° views from a single camera lens into panoramic views.

-   :material-robot: **AI Tracking**
    ---
    Dynamic participant tracking features without moving parts.

-   :material-monitor-dashboard: **Multiple Modes**
    ---
    Original, Discussion, Global, Patrol, and Presentation modes.

</div>
