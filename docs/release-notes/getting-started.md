# Getting Started

Welcome to Moil Meeting! This guide will help you get started quickly.

## Quick Start Guide

1. **Install Moil Meeting** - Follow the installation guide for your operating system
2. **Connect Your Camera** - Ensure your fisheye camera is properly connected
3. **Launch the Application** - Start Moil Meeting from your applications menu
4. **Configure Camera** - Set up your camera parameters in the configuration panel
5. **Start Meeting** - Begin your meeting using the available modes

## First Steps

### Setting Up Your Camera

Before using Moil Meeting, make sure your fisheye camera is:

- Properly connected to your computer
- Recognized by the operating system
- Configured with the correct parameters

### Choosing a Mode

Moil Meeting offers several modes for different scenarios:

- **Original Mode** - View the raw fisheye image
- **Discussion Mode** - Optimized for group discussions
- **Global Mode** - Wide-angle view of the entire room
- **Patrol Mode** - Automated scanning of the room
- **Presentation Mode** - Focus on presenter and slides
