# Moil Meeting Documentation

## Overview

Moil Meeting is an innovative video conferencing solution that leverages MoilApp's high-quality fisheye image processing technology. Built on the Moildev SDK, the system specializes in processing fisheye imagery to produce panoramic and wide-angle (360°) views.

Using a single fisheye camera, Moil Meeting provides comprehensive room coverage and dynamic participant tracking capabilities, delivering a simpler, more flexible, and cost-effective alternative to traditional video conferencing systems.
